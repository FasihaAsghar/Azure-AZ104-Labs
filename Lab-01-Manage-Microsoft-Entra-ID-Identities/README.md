# Lab 01 - Manage Microsoft Entra ID Identities
## Lab Overview

This lab demonstrates the basics of Microsoft Entra ID identity management in Microsoft Azure.

In this lab, I created and configured user accounts, invited an external guest user, created a security group, and managed group membership.

## Objectives

- Understand Microsoft Entra ID and tenant concepts
- Create internal user accounts
- Invite external guest users
- Create Security Groups
- Add users to groups
